# Framer Badges
Unofficial collection of *Framer Badges* for your project's **README** file.

## Behind the Story
When I learning Framer, sometimes I searched source code of a project on Github. I'm always happy when the repository has an explanation and a demo link of the project, but most don't exist, so I have to clone the project into my computer and it's very inefficient. I decided to create this badge so that people who shared their code in Github could link the project interestingly and easily! ðŸ‘Œ

Just copy and paste the code below and replace the link with yours.
